# Dependency
Some libraries have dependencies to others:
- nasm
- pkg-config
- zlib (for ffmpeg output of png format)
- openssl
- cmake
    - requires openssl
- libxml2
- freetype
- fontconfig
    - requires libxml2
- sdl (for ffplay)
- libbluray
    - requires libxml2
    - requires freetype
    - requires fontconfig
- aom
- opensh264
- x264
- x265
- vpx
- lame
- opus
